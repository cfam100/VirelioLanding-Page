# VirelioCreator Landing Page

Landing page for [VirelioCreator](https://creator.virelio.nl) — AI-powered website generator for Dutch businesses. Built as a static site with Tailwind CSS and Three.js.

## Quick Start

```bash
npm run dev
```

## Tech Stack

- Static HTML + Tailwind CSS (CDN)
- Three.js 3D hero scene
- Formspree for contact form
- 5 languages (NL, EN, ES, DE, FR) with browser auto-detect

## Project Structure

```
├── public/
│   ├── index.html      # Full landing page
│   ├── logo.jpeg       # Brand logo
│   ├── robots.txt      # Crawler rules
│   └── sitemap.xml     # Sitemap
├── package.json
└── README.md
```

## Deployment

Static site — deploy the `public/` folder to any host (GitHub Pages, Cloudflare Pages, Netlify).
